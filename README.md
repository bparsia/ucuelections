# UCU Election Data and Explorer

UCU has annual elections for around half of the National Executive Committe (NEC) (and thus half of each sectoral subcommittees, 
the Further Education Committee (FEC) and the Higher Education Committee (HEC)) as well as the (alternating sector) Vice-President and, 
every 5 years, the General Secretary.

Turnout is a problem! I've been worried about it for a [while](https://bparsia.wordpress.com/2021/02/05/some-thoughts-about-democracy/). 
Understanding individual elections and trends is hard because the data is not easily analysable as it's a mishmash of webpages and PDFs 
(and the occasional Word doc). This repo contains tools for extracting the data and to provide analyses thereof

## Contributing

The biggest contribution needed at the moment is data and code review. You can run the extractors yourself or contact me for the CSVs!

I'm happy to take suggestions or contributions for data presentation and analyses!

There's a lot of data and it's hard to aggregate. Turnout can be relative to national eligibility or regional eligibility. Etc.

There's also other data collection. I'm interested in incorporating factional identification, endorsements, branches, etc. I'd love
to incorporate national dispute balloting data, but, turnout is relatively flat (with noise in HE) with spikes for GS elections. So,
these would mostly confirm that "events don't matter much for election turnout".

(Total membership data would be nice, thought eligible voters is probably close enough.)
